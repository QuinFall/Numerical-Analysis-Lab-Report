## 📌 项目简介

本项目是针对《数值分析》第三章 *解线性方程组的直接法* 的完整上机实验实现。
提供一个功能完备的 **GUI 图形化操作系统**，学生可通过输入方程组系数，选择算法后自动完成：

* 系统自动判断方程合法性
* 自动分析矩阵结构并推荐最优求解算法
* 展示求解过程中的每一步矩阵变化
* 展示最终解向量
* 支持 TXT、CSV 导出结果
* 提供良好的交互体验

本项目对教材中涉及的主要直接法进行了完整实现：

| 方法 | 名称             | 特点             |
| -- | -------------- | -------------- |
| ①  | 列主元高斯消元法       | 稳定性较强的基础直接法    |
| ②  | 全主元高斯消元法（主元素法） | 交换行列，稳定性更高     |
| ③  | 克劳特 LU 分解      | 适合一般矩阵，U 对角为 1 |
| ④  | 平方根法（Cholesky） | 适用于对称正定矩阵      |
| ⑤  | 追赶法（Thomas）    | 高效求解三对角线性方程组   |

## 🧠 核心算法说明（简明教材化）

以下对五类算法做简明数学解释，便于课程理解。

---

### **1. 列主元高斯消元法（Partial Pivoting）**

核心思想：

1. 消元过程形如：
   [
   a_{ik} = a_{ik} - \frac{a_{ik}}{a_{kk}} a_{kk}
   ]
2. 每一列都选择绝对值最大的主元
3. 降低舍入误差，提高数值稳定性

代码体现：寻找最大 pivot、交换行、前向消元、回代求解。

---

### **2. 全主元高斯消元法（Full Pivoting）**

比列主元更稳定：

* 在未消元子矩阵中寻找绝对值最大元素作为主元
* **允许行交换 + 列交换**
* 需要通过 `col_perm` 数组记录变量交换，最终还原解变量顺序

适用于对稳定性要求高的矩阵求解。

---

### **3. 克劳特 LU 分解（Crout LU）**

将矩阵分解为：

[
A = L U,\quad U_{ii}=1
]

结构：

* L：下三角
* U：上三角，对角元素恒等于 1

求解步骤：

1. LU 分解
2. 前代求解 ( L y = b )
3. 回代求解 ( U x = y )

优点：适用范围广，易于实现。

---

### **4. 平方根法（Cholesky）**

适用条件：
✔ 对称
✔ 正定

分解形式：

[
A = L L^T
]

步骤：

1. 构造下三角矩阵 L
2. 解 ( L y = b )
3. 解 ( L^T x = y )

复杂度比普通 LU 少一半，稳定性很好。

---

### **5. 追赶法（Thomas Algorithm）**

适用于：

* 三对角矩阵
* 线性复杂度 ( O(n) )

核心思想：

对三对角系统：

[
a_i x_{i-1} + d_i x_i + c_i x_{i+1} = f_i
]

将其前向修正为无回溯项的形式，再从底部回代求解。

非常高效，在工程中常用于差分离散问题。

## 🎯 自动算法推荐机制

通过矩阵结构自动分析：

```python
if is_tridiagonal(A):
    → 推荐追赶法
elif is_symmetric(A):
    → 推荐平方根法
else:
    → 默认推荐列主元高斯消元
```

用户仍可以手动选择其他方法。
推荐信息会在界面中以红色提示展示。
